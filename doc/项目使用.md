### 读取用例

```python
from common.read.readTestcase import readTestcase

# 读取testcase目录下的yaml用例文件;
# 参数 filename:yaml用例文件名,item:yaml文件中的索引,encoding:文件编码
readTestcase("filename", item=0, encoding="utf-8")
```

### 发送请求

```python
from common.request.autoRequest import autoRequest

# 自动请求;
# 参数 caseinfo:读取出的单条用例,timeout:请求超时时间
autoRequest("caseinfo", timeout=10)
```

### 变量

```python
from utils.variablesManager import variables

# 设置变量
# 参数 key:变量名 value:变量值
variables.set("key", "value")
# # 获取变量
variables.get("key")
```

### session

```python
from common.session.sessionManager import session

# 创建session
# 参数 item:创建session数量
session.new(item=1)
# 获取session
# 参数 seek:session指向,默认为第一个session
# 注意:切换seek指向后，不传seek参数时默认为上一次的seek
session(seek=None)
```
